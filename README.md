## Getting Started

### Prerequisites
- Node.js 22+
- Angular CLI 21+

### Installation

```bash
# Clone the repository
git clone https://github.com/etefera11/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
npm install

# Start development server
ng serve
```

Open your browser and navigate to `http://localhost:4200`
